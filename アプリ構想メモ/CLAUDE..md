## コーディング規約
- コメントは日本語でOK

## 削除済み機能（復活させないこと）

| 機能 | 削除理由 |
|------|---------|
| 手書きモード（canvas） | 使われない・重い |
| 制限時間タイマー | 子どもがストレスを感じる |
| タイマーセクション（設定画面） | タイマー削除に伴い |
| 絵文字（UI全般） | 文字のみに統一予定 |

**IMPORTANT: 手書きモード・タイマーは絶対に復活させない**

## 現在の機能（v30）

### データソース
- ミニラボ 166語（Unit1〜10）
- 英検5級 107語
- まぶたん 343語
- オリジナル登録（localStorage）

### 学習モード
- **単語暗記モード**: 日本語→音声→スペルアウトで覚える
- **英単語フラッシュ**: 日本語を見て英語で答える（キーボード入力）
- **英文暗記モード**: 日本語→音声→英文で覚える
- **英文タイピング**: 日本語を見て英文を打つ
- **オリジナル登録**: 一括（テキスト貼り付け）・1件ずつ

### 設定機能
- 出題ソース選択（複数選択可）
- ユニット選択
- 入力方式選択（キーボードのみ）
- スペルアウト音声（ON/OFF・速度選択）
- 説明トグル（toggleSettingDesc）

### 音声機能
- Web Speech API使用
- 英語音声（en-US優先）
- スペルアウト：単語読み上げ→1文字ずつ読み上げ
- ゆっくり（rate:0.55）/ ふつう（rate:0.85）

## よくあるミスと対処法

| ミス | 対処 |
|------|------|
| GitHubアップ後にボタンが押せない | `toggleSettingDesc`が含まれているか確認。Upload filesで上書き |
| テキスト貼り付けでコードが切れる | ファイル添付で渡す |
| キャッシュが残って更新されない | URLの後ろに`?v=2`をつける |
| まぶたんのgridが初期表示される | `display:none`で初期非表示に設定 |
